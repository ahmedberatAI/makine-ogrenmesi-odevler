# odev3Vectors

Bu klasor, `code.ipynb` notebook dosyasini ve notebook'u calistirmak icin gereken temel proje dosyalarini icerir.

## Icerik

- `code.ipynb`: Basit bir Python hucresinde `hello world` yazdiran Jupyter Notebook
- `requirements.txt`: Notebook ortamini kurmak icin gerekli paketler
- `.gitignore`: Gecici ve yerel gelistirme dosyalarini yok sayar

## Kurulum

```bash
pip install -r requirements.txt
```

## Calistirma

```bash
jupyter notebook code.ipynb
```
